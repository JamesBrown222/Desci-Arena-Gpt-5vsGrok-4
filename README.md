**Desci Arena** is a novel project at the intersection of Decentralized Science (DeSci) and DeFi. At its core, it's a live experiment to test the efficacy of Large Language Models (LLMs) in one of the most volatile and fast-paced markets in existence: newly launched memecoins on platforms like pump.fun.

The project deploys two competing AI agents, each with its own trading model, into a perpetual "arena." Their sole objective is to trade new tokens and generate profit.

All net profits from this AI battle are programmatically distributed to the $AIArena token holders and reinvested into the project's development.

## The Core Concept: The Arena

The "Arena" is where our two AI models compete. Here’s how it works:

1.  **The Competitors:** We have deployed two distinct LLM-based trading agents. Think of them as "Agent Alpha" and "Agent Omega." Each has been trained on different datasets and uses unique strategies to identify opportunities.
2.  **The Battlefield:** Both agents constantly monitor the Solana blockchain, specifically focusing on new token listings on pump.fun.
3.  **The Battle:** The AIs analyze real-time data—such as social media sentiment, token distribution, contract integrity, and initial buy momentum—to make high-speed decisions. They autonomously execute buy and sell orders.
4.  **The Spoils:** The goal is simple: generate profit. The bots will trade against each other and the wider market, aiming to capitalize on micro-trends and exit positions before they collapse.

  ## Code Concept Demo

We believe in showing, not just telling. While our production trading bots are private for security, you can run a conceptual demo script locally to see the *core logic* of the Desci Arena in action.

This script simulates the "battle" between our two AI agents, "Alpha" and "Omega," as they analyze a feed of mock memecoins and make competing trade decisions.

### How to Run the Demo:

1.  **Clone the repository:**
    ```bash
    git clone [Your-Repo-URL]

    ```

2.  **Navigate to the demo folder:**
    ```bash
    cd demo
    ```

3.  **Run the Python script:**
    ```bash
    python conceptual_trader.py
    ```

### What This Demo Shows:

* **Competing Strategies:** You can see how `llm_agent_alpha` (our aggressive bot) and `llm_agent_omega` (our cautious bot) have different logic and will make different "BUY" or "PASS" decisions on the same token.
* **Real-Time Simulation:** It mimics a live environment where new tokens appear rapidly.

### How a Real Implementation Differs:

In the live, production version of the Arena:

* `mock_fetch_new_tokens()` is replaced with a **real-time Solana node connection** that monitors the pump.fun contract for new token creations.
* `llm_agent_alpha()` and `llm_agent_omega()` are replaced with **API calls to our fine-tuned LLMs**. They are fed a rich data prompt (token contract, holder distribution, deployer's wallet history, live social sentiment) to make a decision.
* A `BUY` decision triggers a **secure, programmatic transaction** from the bot's trading wallet to the pump.fun contract.
